# React + Vite
A simple and responsive Todo List application that helps users manage daily tasks efficiently. Users can add and delete tasks with an easy-to-use interface.

✨ Features
➕ Add new tasks
🗑️ Delete tasks
📱 Responsive design

🛠️ Technologies Used
HTML
CSS
reactJS

🚀 Getting Started
1. Clone the repository

cd todo-list
3. Install dependencies
npm install
4. Start the application
npm start
